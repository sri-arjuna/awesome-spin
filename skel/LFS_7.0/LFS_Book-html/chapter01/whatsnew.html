1.2. What's new since the last release

Below is a list of package updates made since the previous release of the book.

Note

A somewhat major change was made in this version of the book by adding a new top level directory, /run. This directory has a tmpfs mounted and is used by programs like udev to store run time information. The directories /var/run and /var/lock are also linked to this directory. The bootscripts have been updated to accomodate this change.
