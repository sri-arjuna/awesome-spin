1.3. Changelog

This is version 7.0 of the Linux From Scratch book, dated Oct 29, 2011. If this book is more than six months old, a newer and better version is probably already available. To find out, please check one of the mirrors via http://www.linuxfromscratch.org/mirrors.html.

Below is a list of changes made since the previous release of the book.
